# Цель работы

Закрепить навыки разработки Vue-приложений с использованием современного стека: Pinia, Pinia Persist, vue-router, компонентный подход и архитектурная организация проекта.
В результате выполнения работы студент должен:
- рефакторить существующее приложение To‑Do под хранение состояния через Pinia;
- реализовать авторизацию и регистрацию (без реального бэкенда — имитация);
- внедрить роутинг и создать минимум три страницы;
- оформить творческую страницу «О приложении».

## Технологический стек

- Vue 3 (Composition API)
- Vite — сборка и запуск проекта
- SASS (SCSS) — оформление интерфейса
- JavaScript


## Основные задачи

- Подключить и настроить Pinia
- Реализовать простую систему регистрации и авторизации
- Настроить vue-router
- Страница «О приложении» (About)
